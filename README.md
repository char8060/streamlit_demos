# streamlit_demos
Demos for streamlit.

## Churn Viz
Needs churn_model to run.
To run the churn viz:

```
cd apps/churn/
streamlit run demo.py
```

![demo](img/churn_demo.png)

## Semantic Search Viz

To run the churn viz:

```
cd apps/semantic_search/
streamlit run search_app.py
```
